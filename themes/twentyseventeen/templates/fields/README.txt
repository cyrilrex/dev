Templates for fields are stored in this directory.

Naming conventions for fields will follow this type convention (given a field
named 'field_foo'):

If the field is on (for example) an "article" content type:
field--node--field-foo--article.html.twig
field--node--field-foo.html.twig
field--node--article.html.twig
field--field-foo.html.twig
field--text-with-summary.html.twig
field.html.twig


You can get default templates to work with/override from the 'Classy' theme's
templates folder - /core/themes/classy/templates
